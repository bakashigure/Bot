# plugin: random

*可显示部分开始*

> 随机数

#### 功能

- `ran(dom)|随机(数)`：一个随机小复数（省略部分后同）
- `ran|随机 n`：1 到 n 的随机整数
- `ran|随机 -n`：-n 到 0 的随机整数
- `ran|随机 a b`：a 到 b 的随机整数
- `dy` `Dy` `xdy` 或 `xDy`：x 个 y 数骰的结果

#### 权限

- 开关：管理员
- 使用：全部
